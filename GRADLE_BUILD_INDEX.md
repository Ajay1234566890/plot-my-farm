## ğŸš€ Quick Start (5 Minutes)

### 1. Check Prerequisites
```bash
java -version          # JDK 11+
echo $ANDROID_HOME     # Android SDK
node --version         # Node 16+
```

### 2. Generate Native Project
```bash
npx expo prebuild --platform android --clean
```

### 3. Create Signing Key
```bash
keytool -genkey -v -keystore my-release-key.keystore \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias my-key-alias
```

### 4. Build APK
```bash
cd android
./gradlew assembleRelease
```

### 5. Install
```bash
adb install -r android/app/build/outputs/apk/release/app-release.apk
```

## ğŸ¯ Build Types

### Debug APK (Fast - for testing)
```bash
cd android && ./gradlew assembleDebug
```
- Time: 2-5 minutes
- Size: 50-80 MB
- Use: Testing and development

### Release APK (Optimized - for production)
```bash
cd android && ./gradlew assembleRelease
```
- Time: 5-10 minutes
- Size: 30-50 MB
- Use: Production and distribution

### App Bundle (For Play Store)
```bash
cd android && ./gradlew bundleRelease
```
- Time: 5-10 minutes
- Size: 20-40 MB
- Use: Google Play Store

## ğŸ”‘ Important Security Notes

### Keystore File
- Keep `my-release-key.keystore` safe
- Don't commit to version control
- Add to `.gitignore`
- Back up in secure location

### Keystore Properties
- Keep `android/keystore.properties` safe
- Don't commit to version control
- Add to `.gitignore`
- Contains sensitive passwords

### Environment Variables
- Don't hardcode passwords in scripts
- Use environment variables for sensitive data
- Keep credentials secure
